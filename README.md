# T12
T12固件（STM32）
--------------------------------------- 说明 ----------------------------------------
这是基于STM32F103C8T6开发的T12焊台固件，焊台的PCB文件、PDF版原理图以及外壳的stl文件均在Resources.zip文件里。
该版本是优化后的结果，PCB文件以及固件都进行了优化升级，如发现任何bug可以联系作者修改。
具体的效果可以参考作者上传在bilibili上的视频：https://www.bilibili.com/video/BV1ifQbY2EYM/
